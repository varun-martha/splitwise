## Description

SplitWise is a simple web application for managing shared expenses among friends. This web application is built with **React**, **TypeScript** and **Jest** for testing and this application allows users to track expenses, settle split payments and manage friendships.

## Features

1. **User Registration**:
   - Users can create an account with an email, username and password.
2. **Login**:
   - Secure login with validation.
3. **Add a Friend**:
   - Able to add friends.
4. **Invite a Friend**:
   - Invite friends via email to join in the application.
5. **Add an Expense**:
   - Add shared expenses and split costs among friends.
6. **Settle an Expense**:
   - Mark expenses as settled when payments are completed.
7. **Recent Activity**:
   - Able to see a log of recent activities, including added expenses and settled payments.

## Prerequisites

- **Node.js** (>= 18)
- **npm**
- **MongoDB** (for backend database)
- **React** (Version == 18)
- **TypeScript**
- **Jest** (Testing)

## Usage

1. **Clone the Repository**

   ```sh
    git clone git@github.com:varun-martha/splitwise.git

    cd splitwise
   ```

2. **Install the dependencies**

   ```sh
   npm install
   ```

3. **Run Tests**

   ```sh
   npm run test
   ```

4. **Start the server**
   ```sh
   npm start
   ```
   > **Note : Your development server will start at PORT NO : 3000**
